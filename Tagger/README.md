# NLP-词性标注器
这里主要实现了两种词性标注器，分别基于hmm模型和平均感知器模型，目前只做了英文版本。
### 1.HMM 隐马模型进行英文词性标注
<p>Key Point: 隐马模型，Viterbi解码，数据平滑</p>
<p>Accuracy=0.940300</p>
### 2.Perceptron 感知机词性标注
<p>多类感知机参数矩阵的迭代更新</p>
<p>Key Point: 特征抽取，特征映射，分数计算</p>
